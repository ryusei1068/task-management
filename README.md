![result](https://raw.github.com/wiki/ryusei1068/task-management/images/mcbcc-2byvn.gif)

## Require
Node.js v16.13.1  
Mysql 8.0.28  
  
## Usage
```
$ git clone https://github.com/ryusei1068/task-management.git taskmanagement
$ cd taskmanagement
$ npm install
$ make
```
